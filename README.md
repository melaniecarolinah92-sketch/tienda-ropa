#  Sistema de Tienda de Ropa

![Estado](https://img.shields.io/badge/Estado-En%20Desarrollo-yellow)

##  Descripción

Este proyecto consiste en un sistema de tienda de ropa desarrollado en Python utilizando Tkinter para la interfaz gráfica y MongoDB Atlas como base de datos.

Fue creado con el objetivo de simular el funcionamiento de una tienda real, permitiendo la gestión de usuarios, productos y compras.

###  Motivación
Desarrollar una aplicación completa que integrara interfaz gráfica, base de datos y lógica de negocio.

###  Problema que resuelve
Permite administrar una tienda de ropa de manera digital, facilitando:
- Registro de usuarios
- Inicio de sesión con roles
- Gestión de productos
- Proceso de compra

###  Aprendizaje
Durante el desarrollo aprendí:
- Uso de Tkinter para interfaces gráficas
- Conexión con MongoDB Atlas
- Manejo de usuarios con roles (admin/usuario)
- CRUD completo

##  Características

- Registro de usuarios con imagen
- Inicio de sesión con roles
- Panel de administración
